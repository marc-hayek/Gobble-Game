Gobble-Game
===========

An HTML5 game where you collect junk food and avoid the healthy snacks

Description
-----------

My first attempt at learning html,css,javascript.

Pick a picture for yourself and a picture for the enemy, then go around collecting junk food and avoiding healthy snacks. Use the junk food collected
as weapons to blow up the healthy food items, and later, to defeat the enemy.

Resources
---------

This game uses jQuery as well as the Jaws.js libarary

http://jawsjs.com/

The music used in the game was created by myself.
